# 🛠️ Cyber-Utils

Welcome to **Cyber-Utils**, a centralized collection of scripts and tools developed or curated for various cybersecurity operations.

This repository serves as a personal Swiss Army knife, providing resources for both offensive and defensive security scenarios encountered during CTFs, professional engagements, and research.

## 🔍 Focus Areas

The scripts within this repository cover a wide range of domains, including:

* **CTF & Pentesting**: Automation scripts, exploitation techniques, and post-exploitation modules.
* **OSINT**: Tools for information gathering and digital investigation.
* **Blue Team**: Scripts for monitoring, log analysis, and defensive hardening.
* **Web Security**: Payloads, XSS/SQLi generators, and custom fuzzing utilities.

## 🚀 Getting Started

1. **Clone the repository:**
```bash
git clone https://github.com/S2K7x/Cyber-Utils.git

```


2. **Explore:** Navigate through the directories to find scripts relevant to your current task.
3. **Dependencies:** Ensure you have the necessary runtimes installed (Python, Node.js/TypeScript, PHP, etc.) as indicated by the file extensions within each folder.
